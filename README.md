# AI Code Review Assistant 

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![PyTorch](https://img.shields.io/badge/PyTorch-DeepLearning-red.svg)
![Transformers](https://img.shields.io/badge/HuggingFace-Transformers-yellow.svg)

A deep learning-based AI Code Review Assistant that automatically analyzes source code, identifies potential issues, detects code quality concerns, and generates intelligent review suggestions using transformer-based code embeddings (CodeBERT). The system helps developers improve code quality through automated, context-aware feedback.

---

##  Dataset

- **Dataset:** Code Review Dataset  
- **Task:** Automated Code Review  

---

##  Workflow

- Loaded and preprocessed code review dataset  
- Cleaned and normalized source code snippets  
- Generated contextual embeddings using **CodeBERT**  
- Extracted semantic features from code  
- Built deep learning-based review prediction pipeline  
- Generated automated code review suggestions  
- Evaluated model performance on unseen data  
- Produced intelligent review comments for developers
